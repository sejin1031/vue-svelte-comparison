# vue-svelte-comparison

서버 모니터링 테스트 페이지를 위한 branch

Vue.js 실행
vue/compare-vue 폴더에서 다음 명령어 실행
npm install
npm run serve


빌드 시 : npm run build

Svelte.js 실행
svelte/svelte-app 폴더에서 다음 명령어 실행
npm install
npm run serve

빌드 시: npm run build

express 서버 실행
/socketio-with-express/ 폴더에서 다음 명령어 실행
npm install
node server 를 통해 서버 실행
